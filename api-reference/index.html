API Reference

This page is the entry point to the complete public API documentation.

Collections

  • Seq[T] — Immutable Sequence (tuple-backed)
  • Vec[T] — Mutable Sequence (list-backed)
  • Set[T] — Immutable Set (frozenset-backed)
  • SetMut[T] — Mutable Set (set-backed)
  • Dict[K, V] — Mutable Mapping (dict-backed)
  • Range — Integer ranges (range-backed)

Iterators

  • Iter[T] — Lazy processing of iterators

Error handling

  • Result[T, E] — Actual type to use for explicit error handling (Ok / Err). Used for type hints.
  • ResultType[T, E] — Common interface for Ok and Err. Reference their methods.
  • Ok[T] - Ok variant of Result
  • Err[E] - Err variant of Result
  • ResultUnwrapError - Exception raised when unwrapping a Result that is an Err

Optional values

  • Option[T] — Optional values (Some / NONE)
  • Some[T] - Some variant of Option
  • NONE - NONE variant of Option. Constant singleton.
  • OptionUnwrapError - Exception raised when unwrapping an Option that is NONE

Mixins & ABC's

Fluent Mixins

  • Pipeable — Fluent methods for chaining operations
  • Checkable — Conversion to Option/Result based on instance truthiness

Abstract Base Classes

  • PyoIterable[T] — Base ABC for all iterables
  • PyoIterator[T] — Iterator ABC
  • PyoCollection[T] — Base ABC for eager collections
  • PyoSequence[T] — Sequence ABC
  • PyoMutableSequence[T] — Mutable sequence ABC
  • PyoSet[T] — Set ABC
  • PyoMappingView[T] — Mapping view ABC
  • PyoMapping[K, V] — Mapping ABC
  • PyoMutableMapping[K, V] — Mutable mapping ABC

Mapping Views

  • PyoKeysView[K] — Keys view
  • PyoValuesView[V] — Values view
  • PyoItemsView[K, V] — Items view
